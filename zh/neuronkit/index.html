为什么选择 NeuronKit

上下文即智能

采集设备侧信号并组织为语义/情景/程序性记忆，支持更相关的推理。

安全的设备侧动作

能力模型 + 策略引擎，确保代理触发的动作在执行前得到校验、同意与审计。

对话流式体验

ConvoUI 将自然语言与内嵌 UI 组件结合，用户可“说 + 点”完成任务。

无侵入集成

作为中间件快速接入，最小改动即可把功能注册为能力，由 NeuronKit 负责上下文、策略与 UI。

开发者与企业友好

简单 API + 企业级合规、可观测与治理能力。

关键特性

  • 丰富上下文： 位置、传感器、遥测等作为情景上下文。
  • 设备侧沙箱： 功能 → 能力 → 原语；敏感操作需同意。
  • 云端记忆： 时间序列日志促进为语义/情景/程序性记忆。
  • 代理路由： 将意图路由到注册代理，返回动作计划。
  • ConvoUI： 语音/聊天 + 内嵌 UI 组件。
  • 分析与审计： 策略决策、同意与动作轨迹。
  • 平台无关： 兼容 Android / iOS / IoT，云中立。

适用场景

  • Agentic 电商与超级应用
  • 智能家居与 IoT 控制
  • 现场服务、物流与运营
  • 注重同意与隐私的医金场景

内置防护

  • 高风险操作的人在回路提示
  • 细粒度能力范围
  • 逐次同意与审计追踪

与云端沙箱的对比

维度 云端沙箱（如 E2B/Modal） NeuronKit（设备侧）
工具运行位置 服务器容器 直接在应用 / 设备内
上下文丰富度 受限于设备实时信号 完整设备上下文（受策略控制）
隐私与同意 依赖云侧策略 设备侧策略引擎 + 逐次同意
时延 执行需往返 边缘执行，降低感知时延
体验 仅聊天，UI 分离 聊天 + 内嵌 UI 组件
